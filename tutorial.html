Here is a complete list of instructions (commands) this simulator supports:

Notice the initial number is the decimal command number. It may be entered in place of a command, for instance, the command: 

11

Will increment the Y Register. This value can be entered in hexadecimal if it follows a # character. For instance, the following is the same command: 

#B

This is the same as using the mnemonic abbreviation, for instance, the following is the same code as above: 

INY
